Getting Started

PaladinsPHP

This a wrapper for the Paladins API by Hi-Rez studios.

You can find the PHP Documentation here: https://bennetgallein.github.io/PaladinsPHP

Installing

You can either copy the PHP file directly into your project or preferable just use composer.

composer require bennetgallein/paladins-php

Usage

This library is designed to be as easy to use, as possible, so the start is not that hard. Tho, you should know how to use composer:

<?php

namespace Paladins;

use \Paladins\Paladins;

$devId = 'YOUR_DEV_ID';
$authKey = 'YOUR_DEV_AUTHKEY';
$format = Paladins::JSON; // see Documentation
$lang = Paladins::ENGLISH; // see Documentation

$paladins = new Paladins($devId, $authKey, $format, $lang);

$paladins->connect();
?>

NOTE: The API uses APCu to save the session. Make sure to have it installed! Instructions are here: php.net

Language and Format:

This library offers a few constats which are definetly good to use!

Language Codes:

const ENGLISH = 1;
const GERMAN = 2;
const FRENCH = 3;
const SPANISH = 7;
const SPANISHLA = 9; // Spanish (Latin America)
const PORTUGUESE = 10;
const RUSSIAN = 11;
const POLISH = 12;
const TURKISH = 13;

Response Formats:

const JSON = 'Json';
const XML = 'xml';

Access them with: Paladins::<NAME> for example: Paladins::ENGLISH;


Important for connecting!

The Basic function to connect with the API-Endpoint. It takes 4 Parameters, not more, not less. $devId, $authKey, $format, $lang, where $format is the response format of the API and $lang the language you want to use.
